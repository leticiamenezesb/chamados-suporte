# Chamado 5

**Título:** Erro: Arquivo não encontrado (404)  

**Descrição do problema:**  
O link acessado não existe ou foi movido.  

**Passos para reproduzir:**  
1. Abrir o sistema  
2. Tentar executar a ação que gera o erro  
3. Capturar a tela com a mensagem exibida  

**Resultado esperado:**  
A ação deve ser concluída sem apresentar erro.

**Resultado atual:**  
Mensagem de erro: **Erro: Arquivo não encontrado (404)**
